# SaaS / On-Prem Point 34 - Executive Response

This response should be interpreted together with [01-assessment-context.md](./01-assessment-context.md).

IGS Billing provides implementation, administration, and usage documentation appropriate to its managed-service model. The solution is hosted in IGS infrastructure and made available to GS via web access protected by VPN, controlled URL exposure, and IP allowlisting.

The current evidence package includes installation and deployment guidance, administration and hardening documentation, API and authentication usage documentation, and operational validation guidance.

For that reason, the most defensible response is **compliant**.

Recommended questionnaire wording:

"The provider maintains implementation, administration, and usage documentation for the solution under a managed-service model. The application is hosted in IGS infrastructure and made available to GS through secure web access protected by VPN, controlled URL exposure, and IP allowlisting. Available documentation includes deployment and hardening guides, administration procedures, and user/API usage documentation. Therefore, this requirement is currently considered compliant."
